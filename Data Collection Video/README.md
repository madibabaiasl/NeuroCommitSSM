# NeuroCommitSSM Data Collection Video

This folder contains the data collection media for **NeuroCommitSSM**, including task demonstration videos and a reference image used during the experimental protocol.

## Overview

These files document the task conditions used during data collection for the NeuroCommitSSM project. The videos provide a visual reference for how each task trial was performed, and the image provides the resting-state reference used in the protocol.

This folder is intended to help readers, reviewers, and collaborators better understand the experimental setup and the task execution procedure used to acquire multimodal data.

## Files

The following files are included in this folder:

- `Rest.jpg` — reference image for the resting condition  
- `T1.mp4` — task video for Task 1  
- `T2.mp4` — task video for Task 2  
- `T3.mp4` — task video for Task 3  
- `T4.mp4` — task video for Task 4  
- `T5.mp4` — task video for Task 5  

## Purpose

These media files are provided to:

- illustrate the task protocol used during data collection  
- show the physical execution of each experimental task  
- provide visual context for the recorded EEG, EMG, and eye-tracking data  
- support reproducibility and clearer understanding of the NeuroCommitSSM workflow

## Description

### `Rest.jpg`
This image represents the resting-state condition used during the experiment. It serves as a reference for the baseline or non-action phase of the protocol.

### `T1.mp4` to `T5.mp4`
These videos correspond to the task-specific trials used during data collection. Each file provides a visual example of the task performance associated with that task label.

The task videos are included to show how the participant interacted with the experiment during recording and to provide context for the multimodal signals collected in the study.
